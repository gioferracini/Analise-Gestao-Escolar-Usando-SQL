# 🏫 Análise de Gestão Escolar — Projeto SQL
*SQL • MySQL • Análise de Dados*

![Header](Imagens/header.png)

Este projeto simula um sistema de gestão escolar, utilizando SQL para explorar indicadores acadêmicos, perfil dos alunos, desempenho das turmas e estrutura docente. Foram desenvolvidas 25 análises voltadas ao acompanhamento do desempenho educacional, evasão escolar, frequência, matrículas e distribuição de professores.

Todos os dados são fictícios e foram criados exclusivamente para fins de estudo e composição de portfólio.

---

## 🧠 Principais Habilidades

* SQL para análise de dados
* Joins, CTEs e funções de janela
* Modelagem relacional
* Desenvolvimento de KPIs educacionais
* Análise exploratória de dados
* Storytelling com dados
* Interpretação de indicadores acadêmicos

---

## 📂 Estrutura da Base de Dados

O projeto é composto pelas seguintes tabelas:

* **alunos**: informações cadastrais e histórico escolar;
* **turmas**: divisão das turmas por ano e período;
* **disciplinas**: matérias oferecidas pela instituição;
* **professor**: cadastro dos professores;
* **professor_disciplina**: relação entre professores e disciplinas;
* **historico**: notas, frequência e situação acadêmica dos alunos.

---

## 📊 Principais Análises Desenvolvidas

### 📈 Desempenho Acadêmico

* Ranking de alunos por média final;
* Reprovações por disciplina;
* Média das notas por matéria;
* Evolução do desempenho ao longo dos anos;
* Relação entre frequência e aprovação.

### 🎒 Matrículas e Retenção

* Evolução das matrículas por ano;
* Idade média dos alunos;
* Taxa de evasão escolar;
* Quantidade de alunos por turma;
* Distribuição por faixa etária.

### 👨🏻‍🏫 Corpo Docente

* Professores responsáveis por múltiplas disciplinas;
* Carga horária total dos professores;
* Professores ativos por período;
* Distribuição por gênero e faixa etária;
* Quantidade de turmas coordenadas.

### 📚 Indicadores das Turmas

* Quantidade de alunos por turma;
* Períodos mais movimentados;
* Comparação de desempenho entre turmas;
* Relação entre série, idade média e ano letivo;
* Turmas com maior índice de aprovação.

### 🔎 Análises Complementares

* Ranking de frequência dos alunos;
* Disciplinas com maior índice de reprovação;
* Bairros com maior taxa de aprovação;
* Alunos com alta frequência e baixo desempenho.
